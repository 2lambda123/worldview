Class: Worldview.JSON.MapConfig.Projection
Defines a map projection to be available within Worldview.

Any property used to create an OpenLayers.Map object can be provided in this
object. The properties listed below are properties specific to Worldview.

Example:
(begin code)
"geographic": {
    "projection": "EPSG:4326",
    "name": "geographic",  
    "maxExtent": [-180, -180, 90, 90],
    "numZoomLevels": 9,
    "resolutions": [
        0.5625, 0.28125, 0.140625,
        0.0703125, 0.03515625, 0.017578125,
        0.0087890625, 0.00439453125, 0.002197265625
    ],
    "startCenter": [0, 0],
    "startZoom": 1    
}
(end code)

Property: name
The name of this projection used in Worldview to reference this projection. 
(i.e. "geographic", "arctic", "antarctic").

Property: startCenter
Starting map position for this projection. Array containing the latitude and 
longitude values in the units of the projection.

Property: startZoom
Starting zoom level for this projection.
