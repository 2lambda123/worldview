Class: Worldview.JSON.MapConfig.Product
Defines a product available within Worldview.

Any property used to create an OpenLayers.Layer object can be provided in this
object. Properties specific for a certain projection can be placed in the 
<projections> property and these will be copied over and overrides any 
property defined in the general section. 

The properties listed below are the required properties.

Example:
(begin code)
"MODIS_Terra_CorrectedReflectance_TrueColor": {
    "layer": "MODIS_Terra_CorrectedReflectance_TrueColor",
    "productClass": "Worldview.Map.DailyProduct",
    "layerClass": "OpenLayers.Layer.WMTS",
    
    "name": "Corrected Reflectance (True Color)",
    "description": "Terra / MODIS",

    "defaultLayer": "true",
    "start_date": "2012-04-07",
    "format": "image/jpeg",
    "tileSize": [ 512, 512 ],

    "projections": {
        "geographic": {
            "url": [
                "http://map1a.vis.earthdata.nasa.gov/wmts-geo/wmts.cgi",
                "http://map1b.vis.earthdata.nasa.gov/wmts-geo/wmts.cgi",
                "http://map1c.vis.earthdata.nasa.gov/wmts-geo/wmts.cgi"
            ],
            "maxExtent": [-180, -90, 180, 90],
            "matrixSet": "EPSG4326_250m",
            "maxResolution": 0.5625
        },
        "arctic": {
            "url": [
                "http://map1a.vis.earthdata.nasa.gov/wmts-arctic/wmts.cgi",
                "http://map1b.vis.earthdata.nasa.gov/wmts-arctic/wmts.cgi",
                "http://map1c.vis.earthdata.nasa.gov/wmts-arctic/wmts.cgi"
            ],
            "maxExtent": [-4194304, -4194304, 4194304, 4194304],
            "matrixSet": "EPSG3995_250m",
            "maxResolution": 8192.0
        },
        "antarctic": {
            "url": [
                "http://map1a.vis.earthdata.nasa.gov/wmts-antarctic/wmts.cgi",
                "http://map1b.vis.earthdata.nasa.gov/wmts-antarctic/wmts.cgi",
                "http://map1c.vis.earthdata.nasa.gov/wmts-antarctic/wmts.cgi"
            ],
            "maxExtent": [-4194304, -4194304, 4194304, 4194304],
            "matrixSet": "EPSG3031_250m",
            "maxResolution": 8192.0                
        }
    }
}
(end code)

Property: layer
Identifier used to reference the layer from the remote map service.

Property: productClass
The Product object to create to manage the layers for this product. Either
<Worldview.Map.DailyProduct> or <Worldview.Map.StaticProduct>.

Property: layerClass
The OpenLayers.Layer class used to create layers for this product as needed.

Property: name
Descriptive name of this product, displayed to the end user in Worldview.

Property: description
Additional information about this product, displayed to the end user in
Worldview.

Property: defaultLayer
Set to "true" if this layer should be selected when the application starts.

Property: start_date
First date that data is available for this product as a string in YYYY-DD-MM 
format.

Property: format
MIME type of the image to be returned from the map server (e.g., image/jpeg,
image/png)

Property: tileSize
Array of width and height of the tile, in pixels, to request from the
map server.

Property: url
Either a string or an array of strings containing URLs to the map service
endpoint.

Property: maxExtent
Array containing the minimum longitude, minimum latitude, maximum longitude,
and maximum latitude for all data available for this product. Must be 
specified using projected units.

Property: matrixSet
The matrix set to request from the remote server.

Property: maxResolution
The maximum resolution available for this product.

Property: projections
An object containing any of the listed properties above that are specific
to a certain projection. The object should be keyed by the projection name.

Example:
(begin code)
"projections": {
    "geographic" {
        "maxExtent": [-180, -90, 180, 90]
    },
    "arctic": {
        "maxExtent": [-4194304, -4194304, 4194304, 4194304]
    }
}
(end code)
