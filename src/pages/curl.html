Using cURL to Bulk Download Your Data

Add the netrc entries listed above to the following file:

  • Windows: _netrc file found in your home folder.
  • Mac or Linux: .netrc file found in your home directory.
Replace URS_USER with your URS user name and URS_PASSWORD with your URS password. If this file does not exist, create a new one. Ensure that your are the only one that can read this file.

Open a terminal or command prompt, navigate to the directory or folder where you want to save the data, and cut and paste the Download Commands above.
