Required Dependencies
=====================
 
Run the "download-deps" script to download and unpack all dependencies.

- YUI Compressor, Version 2.4.7 
	https://github.com/yui/yuicompressor

- YUI Compressor Ant Task, Version 0.5
	http://code.google.com/p/javaflight-code/wiki/YuiCompressorAntTask

- JSDoc 3, Version 3.0.1
	https://github.com/jsdoc3/jsdoc

- JSDoc 3 Ant Task
	https://github.com/jannon/jsdoc3-ant-task


Targets
=======

To generate the web site in the build/worldview directory:

$ ant web

To generate documentation in the build/doc directory:

$ ant doc

To generate distribution artifacts in the dist directory:

$ ant dist

To clean up:

$ ant clean
